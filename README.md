# 🏥 CareSplit - Built by Team MedAxis | Enyata × Interswitch Buildathon 2026

---

Introduction

CareSplit is a health-fintech web application designed to address one of the most critical challenges in healthcare access across Nigeria and similar markets: the inability of patients to pay large medical bills upfront.

In many healthcare systems, treatment is often delayed or denied until full payment is made. However, most individuals earn income gradually (monthly or irregularly), creating a mismatch between how healthcare is billed and how people earn.

CareSplit bridges this gap by enabling patients to split hospital bills into structured, manageable installment payments, allowing them to access care immediately while paying over time.

---

Product Vision

To build a trusted healthcare payment infrastructure that removes financial barriers to treatment by aligning medical payments with real income patterns.

---

Problem Context

Research conducted by Team MedAxis shows that:

- A majority of individuals pay healthcare costs out-of-pocket
- Many patients delay or skip treatment due to cost
- Large one-time payments create financial stress and instability
- There is little to no formal installment system for healthcare payments

This results in a cycle of:
Medical need → Cost barrier → Delay or financial shock → Debt → Vulnerability

---

Our Solution

CareSplit provides a platform where:

- Hospitals can issue digital medical bills
- Patients can select flexible installment plans (e.g., 3, 6, 12 months)
- Payments are processed through secure, automated systems
- Users can track their payment progress in real time

The system transforms healthcare payments from a single high-cost event into a structured financial journey.

---

Core Features (MVP)

The CareSplit MVP focuses on delivering a working end-to-end experience:

1. User Authentication

- Secure sign up and login
- Account verification (OTP)

2. Bill Management

- View hospital bills
- Access bill details and breakdown

3. Payment Plan Selection

- Choose structured installment options
- Dynamic calculation of monthly payments

4. Payment Authorization

- Select payment method (card/bank)
- Authorize recurring payments

5. Payment Tracking Dashboard

- View remaining balance
- Track payment progress
- Access transaction history

6. Plan Management

- Pause payments
- Adjust installment plans

---

🔄 How It Works (User Flow)

1. User signs up and logs in
2. User views assigned hospital bill
3. User selects a preferred payment plan
4. User authorizes payment method
5. Payment plan is activated
6. User tracks progress and manages payments

System Overview

CareSplit operates as a two-sided platform:

Patient Side

- Access bills
- Choose payment plans
- Manage and track payments

Provider Side (Future Prioritization)

- Create and assign bills

- Track patient payment status

Trust & Security

Given the sensitivity of healthcare and financial data, CareSplit prioritizes:

- Transparent billing and payment schedules
- User-controlled payment actions (pause/reschedule)
- Secure payment processing (via Interswitch infrastructure)
- Clear consent before recurring payments
